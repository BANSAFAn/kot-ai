# Примеры команд для KOT.AI

## Системные команды
- "Открой браузер" - открывает браузер по умолчанию
- "Запусти калькулятор" - запускает калькулятор
- "Открой блокнот" - запускает блокнот
- "Открой сайт google.com" - открывает указанный сайт в браузере по умолчанию
- "Сделай скриншот" - делает снимок экрана и сохраняет его
- "Информация о системе" - показывает информацию о системе
- "Список процессов" - показывает список запущенных процессов
- "Завершить процесс chrome" - завершает указанный процесс

## Управление звуком
- "Увеличь громкость" - увеличивает громкость системы
- "Уменьши громкость" - уменьшает громкость системы
- "Установи громкость 50" - устанавливает громкость системы на 50%
- "Выключи звук" - отключает звук
- "Включи звук" - включает звук

## Управление ассистентом
- "Помощь" - показывает список доступных команд
- "Настройки" - открывает настройки ассистента
- "Выключись" - завершает работу ассистента
- "Перезапустись" - перезапускает ассистента

## Общие вопросы
- "Который час?" - показывает текущее время
- "Какая сегодня дата?" - показывает текущую дату
- "Какая погода?" - показывает информацию о погоде (требуется настройка API)
- "Расскажи анекдот" - рассказывает случайный анекдот

## Примечание
Вы можете добавлять свои команды и расширять функциональность ассистента, модифицируя код в директории internal/assistant/assistant.go.
